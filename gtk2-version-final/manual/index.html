  1. Exe entry

    It's here that you put your executable name.

  2. Main entry

    It's here that you put the name of your main source code. (without the extention)

  3. Main entry

    It's here that you choose the number of library of your current project
    and then you put in the boxes the name of each libraries. (without the extention)

  4. gtk mode

    It's here that you can choose if your makefile can compile your project with gtk flags.

  5. separates libs mode

    It's here that you can choose if your makefile can create differents dynamics libs (lib.a) for each libraries.

  6. commune lib mode

    It's here that you can choose if your makefile can create a dynamic commune lib (lib.a) for each libraries.

  7. check files mode

    It's here that you can choose if the program checks if all the libs and the main exist in your repertory.

  8. compressed mode

    It's here that you can choose if the your makefile can create a tar.gz archive to compressed your project.

  9. custom cflags mode

    It's here that you can choose if your makefile can compile your project with your custom cflags.
    Ther is default cflags. They contains all the warnings for the c language

  10. children mode

    It's here that you can choose if your makefile can compile a makefile in an other repertories that you write the name in the box.

  11. open app mode

    It's here that you can choose if your makefile can open your project with an app that you write the name in the box.

  12. rapport mode

    It's here that you can generate a rapport about the generation of your Makefile.

  13. editor mode

    It's here that you can generate additional instructions at the end of your makefile.
